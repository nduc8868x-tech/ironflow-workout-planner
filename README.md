# IronFlow – Gym Workout Planner

## Overview

IronFlow is a workout planning and tracking application designed to help users:

* Plan workout schedules
* Track exercises (sets, reps, weight)
* Monitor training progress

This project is built following a structured development workflow:
Research → Planning → Requirements → Development

---

## Goals

* Help users stay consistent with their workout routine
* Provide a clean and intuitive tracking experience
* Focus on simplicity and usability

---

## Project Approach (Plan Storm)

This project follows a structured process:

1. Research – Identify user problems and competitors
2. Planning – Define features and user flow
3. Requirements – Specify frontend, backend, and database
4. Development – Implement the system

All documentation is available in the `/docs` folder.

---

## Features

### MVP (v1)

* Create workout schedules
* Add exercises to workouts
* Track sets, reps, and weight

### Future Versions

* Progress analytics
* Workout reminders

## Installation

### Clone repository

```bash
git clone https://github.com/your-username/ironflow-workout-planner.git
cd ironflow-workout-planner
```

### Setup Backend

```bash
cd backend
npm install
npm run dev
```

### Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
